# Security and Access Control Policy

## Access Credentials
- Key fob for building entry
- Mailbox key
- Unit keys (2 sets provided)
- Parking access card (if applicable)

## Lost Credentials
- Report immediately to prevent unauthorized access
- Replacement fees: $75 key fob, $25 mailbox key, $15 unit key
- Locks re-keyed if multiple keys lost: $200

## Guest Access
- Temporary codes valid 24 hours
- Maximum 5 guest codes per month
- Resident responsible for guest entry times

## Security Cameras
- Common areas monitored 24/7
- Footage retained 30 days
- Available for incident investigations
- No monitoring of private unit areas

## After-Hours Access
- Front desk closes at 10 PM
- Emergency maintenance: (555) 123-4567
- Security patrol: 10 PM - 6 AM

## Tailgating
- Do not allow unknown persons to follow you through secured doors
- Report suspicious persons to security
- Violations compromise community safety
