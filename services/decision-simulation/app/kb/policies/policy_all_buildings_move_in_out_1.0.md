# Move-In and Move-Out Policy

## Move-In Process
1. Final walkthrough with property manager
2. Sign condition report
3. Receive keys and access credentials
4. Set up utilities in your name
5. Provide renter's insurance proof

## Move-In Checklist
- Document existing damage (photos/video)
- Test all appliances
- Check smoke/CO detectors
- Report issues within 48 hours

## Move-Out Notice
- 60 days written notice required
- Email to: moveout@apartmentmanager.com
- Must specify exact move-out date
- Early termination fees apply if notice not given

## Move-Out Process
1. Schedule pre-move-out inspection (optional)
2. Remove all belongings
3. Clean unit thoroughly
4. Return all keys and access devices
5. Provide forwarding address
6. Final walkthrough with staff

## Deposit Return
- Itemized statement within 30 days
- Refund mailed to forwarding address
- Deductions for: unpaid rent, damages beyond normal wear, cleaning if needed

## Elevator Reservation
- Required for large furniture/appliances
- Book 48 hours in advance
- Available times: 9 AM - 7 PM
- $200 deposit (refunded if no damage)
