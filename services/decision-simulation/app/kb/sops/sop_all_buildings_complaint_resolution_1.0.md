# Resident Complaint Resolution

## Purpose
Standardize resident complaint resolution to ensure consistency and quality.

## Scope
All property staff involved in complaint resolution.

## Procedure

### Step 1: Preparation
- Review requirements and documentation
- Gather necessary materials and tools
- Schedule appropriate time and resources

### Step 2: Execution
- Follow checklist and guidelines
- Document all actions taken
- Maintain communication with residents

### Step 3: Quality Check
- Verify all tasks completed
- Inspect work quality
- Obtain necessary approvals

### Step 4: Documentation
- Complete required forms
- Update system records
- File supporting documentation

### Step 5: Follow-up
- Confirm resident satisfaction
- Address any outstanding issues
- Schedule future actions if needed

## Quality Standards
- Completion accuracy: 95%
- On-time delivery: 90%
- Resident satisfaction: 4.0/5.0
